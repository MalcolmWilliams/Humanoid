Note on the naming format:
SUBSECTION_NUMBER_COLOR_GRAMS.gcode

subsection: what part of the overall robot, ie skull and ears is SE
number: if there are multiple prints per subsection just give them a serial number
color: b = black, w = white, x = dont care (generally for test parts only)
grams: how many grams of plastic. useful when near end of roll or for estimating time. time estimates are also provided in gcode metadata.
